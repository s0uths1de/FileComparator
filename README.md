[FileComparator](https://github.com/s0uths1de/FileComparator/blob/main/README.md)

快速比对作业查询的GUI界面工具

环境：Liberica JDK Full 64bit [链接](https://www.injdk.cn/)

使用说明：

1、读信息文件
> 读取文件建议为vsc，每一行必须包括10个数字的学号，以及学号

2、读取作业
> 读取一个文件夹，其中为所有人的作业

3、点击开始
> 出现为完成作业的人，命名错误的人（此功能暂未实现）
